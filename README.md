## Build Setup

```bash

From Node v14.21.1

# install dependencies
$ yarn install

# Developement mode with hot reload
$ yarn dev

# Generate static project for production
$ yarn generate
# Launch server
$ yarn start

```

For detailed explanation on how things work, check out the [documentation](https://nuxtjs.org).
